# Formatting Code

If a single statement becomes too large, you can make it span multiple lines by either surrounding the statement with parentheses or using the line-continuation character, which is a `\`.

```{code-cell} ipython3
(2 +
3)
```

```{code-cell} ipython3
2 + \
3
```
